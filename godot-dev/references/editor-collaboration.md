# 编辑器协作协议

采用轻量磁盘协作，不要求 MCP 或编辑器插件。

1. 人保存 Agent 将修改的重叠脚本、场景和资源。
2. Agent 读取保存后的磁盘内容并记录目标范围。
3. Agent 修改文本，运行契约、加载、实例化和项目测试。
4. 编辑器重新扫描外部变化并重新打开或重载受影响场景。
5. 检查本次操作产生的 Output、Debugger、插件状态和运行日志。

普通脚本、资源和场景修改通常只需扫描与重载。AutoLoad、全局类、插件、GDExtension、导入缓存或已删除脚本变化必须完整重启。

如果编辑器提示磁盘与未保存缓冲区冲突，停止操作，由人选择保留哪一版本。Agent 不覆盖无法确认的未保存内容。
